Hello!
This is short description for xCRUD archive.


1. HOW TO INSTALL DEMO
 - Extract this archive on your server.
 - Create new database and Install demo tables:
	- xcrud1.6.sql - basic database.
	- meta_location.sql - locations table for 'Dependent dropdowns' demo.
	- million.sql.zip - table for 'One million rows' demo.
 - Configure database connection for xCRUD ( /xcrud/xcrud_config.php ).
 - Try it:
	- http://your_server_url/demos - main demos
	- http://your_server_url/example.php - the simpliest example


2. WHAT FOLDERS IS IN ARCHIVE
 - demos - all examples is in this folder
 - documentation - latest docs
 - editors - free editors, used for demos
 - integration - examples for integration with other system (currently for CodeIgniter only)
 - Joomla! 2.5 - plugin for easy Joomla! 2.5 integration
 - Joomla! 3.2 - plugin for easy Joomla! 3.2 integration
 - uploads - default uploads folder, used for demo
 - WordPress - plugin for easy WordPress integration
 - xcrud - MAIN APPLICATION
